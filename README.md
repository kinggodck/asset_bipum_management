# (주)영광 비품 QR 관리 프로그램

Google 스프레드시트의 자산관리대장과 개정된 `자산·비품 Naming Rule 및 QR 스티커 관리 절차서` 구조에 맞춘 정적 웹앱입니다.

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.

로컬 서버로 확인하려면:

```bash
npm run dev
```

## Vercel 배포

이 프로젝트는 정적 웹앱으로 Vercel에 바로 배포할 수 있습니다. 배포 시 내부 문서 파일은 제외하고 `dist/`에 생성되는 앱 파일만 공개됩니다.

```bash
npm run build
vercel
```

운영 배포:

```bash
vercel --prod
```

처음 배포할 때 Vercel CLI가 프로젝트 연결을 묻습니다. 기존 Vercel 프로젝트가 없다면 새 프로젝트로 연결하면 됩니다.

배포 산출물만 미리 확인하려면:

```bash
npm run build
npm run preview
```

## 주요 기능

- Google Sheet CSV 동기화
- CSV 파일 가져오기와 내보내기
- 전체 비품, 사용 중, 수리 중, 구매금액 KPI
- 표준 Naming Rule 기반 비품번호 생성: `[자산구분]-[품목]-[도입연월]-[순번]`
- 사업장, 위치, 사용부서 등 변동 정보는 번호에 포함하지 않고 상태 값으로 관리
- 비품번호, 품명, 모델명, 시리얼번호 통합 검색
- 상태, 관리부서, 자산구분 필터
- 비품 상세 보기
- 로컬 비품 등록
- 상태 변경과 최종 실사일 기록
- QR 스티커 미리보기와 인쇄
- 단축 URL 기반 QR 값 생성
- 인쇄업자 전달용 스티커 CSV 다운로드
- IT 기기와 고가 장비용 VOID/파괴 라벨 재질 기준 반영

## Google Sheet 연동 참고

브라우저에서 직접 Google Sheet를 읽으려면 시트의 CSV 접근 권한이 열려 있어야 합니다. 권한 또는 CORS 문제로 동기화가 막히면 Google Sheet에서 CSV로 다운로드한 뒤 `CSV 가져오기`를 사용하면 됩니다.

이 앱의 변경 사항은 브라우저 로컬 저장소에 보관됩니다. Google Sheet에 다시 쓰는 기능은 API 인증이 필요한 별도 서버 또는 Apps Script 연동이 필요합니다.
