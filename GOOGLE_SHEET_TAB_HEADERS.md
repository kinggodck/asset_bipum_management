# Google Sheet 탭 생성용 헤더

아래 3개 탭을 Google Sheet에 직접 추가한 뒤, 각 탭의 1행 A1 셀에 해당 헤더를 붙여넣으면 됩니다.

## 비품마스터

```tsv
비품번호	자산구분코드	품목코드	도입연월	순번	품명	현재상태	현재사용자	사용자소속	사용시작일	반납예정일	사용목적	사업장코드	상세위치코드	관리부서	제조사	모델명	시리얼번호	구매일	구매금액	사진링크	QR_URL	스티커크기	스티커재질	인쇄수량	최종실사일	최근상태변경일	비고
```

## 사용이력

```tsv
이력ID	비품번호	처리유형	처리일시	사용자	사용자소속	사용목적	사용시작일	반납예정일	실제반납일	처리자	이전상태	변경상태	위치	비고
```

## 실사이력

```tsv
실사ID	비품번호	실사일시	실사자	확인위치	확인상태	결과	사진링크	조치내용	소명담당자	소명기한	비고
```

## 권장 드롭다운 값

### 비품마스터 / 현재상태

```text
사용 가능
사용 중
반납
수리 중
폐기 예정
폐기
```

### 사용이력 / 처리유형

```text
사용 시작
이동
대여
반납
수리
폐기
```

### 실사이력 / 결과

```text
정상
```
